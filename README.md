# 🫙 Kombucha Brewing Journal

A simple, elegant brewing log for home kombucha brewers. Track every batch from first fermentation to the final taste — no account, no app, no internet required.


**[Open the journal →](https://sonjaluja.github.io/kombucha-journal)**

---

## Features

- **Dashboard** with brewing statistics, recent batches, best batch, and flavor history
- **First fermentation (F1)** tracking — tea type, sugar, water, SCOBY health, pH, temperature, and duration
- **Second fermentation (F2)** tracking — multiple flavor additions, multiple bottle sizes, carbonation notes
- **Taste ratings** for sweetness, acidity, carbonation, and overall impression
- **Sugar calculator** — enter your batch size in liters and get instant sugar and starter liquid amounts
- **Built-in recipe guide** with tea recommendations, fermentation timing tips, and flavor notes from real brewing experience
- **Export / import** — back up your data as a JSON file and restore it on any device
- **Works offline** — open the file in any browser, no internet needed after the first load

---

## Getting Started

1. Download `index.html`
2. Open it in any modern browser (Chrome, Firefox, Safari)
3. Start logging your first batch

That's it. No installation, no account, no server.

---

## How data is stored

All data is saved in your browser's `localStorage` — it stays on your device and is never sent anywhere. Each person who uses their own copy of the file gets their own completely separate journal.

To keep your data safe or move it to another device, use the **Export** button in the top right corner. This downloads a `.json` backup file you can import again at any time.

## Recipe reference (built into the app)

The dashboard includes a full recipe guide based on real brewing experience:

**Base recipe:** 0.5–0.6 dl sugar per liter of tea · ~100 ml starter liquid per liter

**Recommended teas:** Black (best), green (great), oolong (good), white (ok). Avoid flavored or herbal blends.

**Fermentation timing:**
| Goal | F1 | F2 |
|---|---|---|
| More bubbles, lighter taste | 10 days | 3–4 days |
| More flavor, fewer bubbles | 4–7 weeks | 3–4 days |

**Top flavor additions for F2:** Raspberry, ginger, pomegranate, cherry, blueberry, mixed berries, peach, orange. Frozen fruit that has been thawed works best.
